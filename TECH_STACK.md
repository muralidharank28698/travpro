# Technical Stack

The Zytravo Trvls platform is built using a modern, performance-oriented stack focused on speed, type-safety, and premium user experience.

## Core Frameworks
- **[Next.js 16](https://nextjs.org/)**: Utilizing the App Router for fast, server-first rendering and localized routing.
- **[React 19](https://react.dev/)**: Featuring Concurrent Mode and advanced Hooks for smooth UI interactions.
- **[TypeScript](https://www.typescriptlang.org/)**: Ensuring strict type-safety across the entire codebase.

## Styling & UI
- **[Tailwind CSS v4](https://tailwindcss.com/)**: A next-generation styling engine with the "Emerald Reserve" custom design system.
- **[Lucide React](https://lucide.dev/)**: A comprehensive, lightweight icon set.
- **[next-themes](https://github.com/pacocoursey/next-themes)**: Powering the seamless Dark and Light mode transitions.

## Data & State Management
- **[Supabase](https://supabase.com/)**: Providing the PostgreSQL database, Authentication, and Row Level Security (RLS).
- **[Redux Toolkit](https://redux-toolkit.js.org/)**: Managing complex client-side state for bookings and user preferences.
- **[date-fns](https://date-fns.org/)**: Handling reliable date and time formatting across timezones.

## Globalization (i18n)
- **[next-intl](https://next-intl-docs.vercel.app/)**: Orchestrating full-site translations and localized routing for English, French, and Spanish.
